Terradue::Portal::Catalogue¶

Represents a metadata catalogue.

The catalogue follows the schema adopted by the Terradue Access (CAS) and other metadata catalogues in which entries are catagorised as series and datasets and organised using virtual URLs that make their relationship clear. A typical CAS provides RESTful interfaces for adding, altering or deleting series and data sets where the URLs are organised as <base-url>[/<series-name>[/<dataset-name>]]/<format>. The class follows this model. Note that there is ongoing confusion between the terms series (or dataset) on one side and data set (or product) on the other side.

Properties¶

Type Name Summary
string OpenSearchDescriptionUrl Gets or sets the URL of the top-level OpenSearch document.
string BaseUrl Gets or sets the base URL part that precedes all relative URLs to series or datasets.
string SeriesIngestionUrl Gets or sets the REST URL at which new series are ingested.
